## 1. 前置条件

- [ ] 能访问 125 生产机（或等效 Windows Server）
- [ ] MySQL 8.0+ 已运行
- [ ] JDK 17 + Maven 3.8+ 已安装
- [ ] 生产 `D:\archive\config\config.json` 已配置（含 GLM API key、JWT secret）
- [ ] **备份当前数据库**：`mysqldump -u root archive_db > D:\backup\archive_db_$(date +%Y%m%d).sql`

## 2. 升级步骤

### 2.1 拉取最新代码

```bash
cd D:\projects-online
git checkout main
git pull origin main
```

### 2.2 执行数据库迁移

```bash
cd deploy/sql
mysql -u root archive_db < migrate_260611_01.sql
# 期望输出: 0 错误，112+ queries OK
```

**验证**：

```bash
mysql -u root archive_db -e "DESC notification;"
# 期望: is_read TINYINT(1) (不是 `read`)
mysql -u root archive_db -e "SELECT COUNT(*) FROM role;"
# 期望: 6 (admin/committee/pm/user/legal/secretary)
```

### 2.3 构建后端

```bash
cd D:\projects-online\backend
mvn clean package -DskipTests -B
# 期望: BUILD SUCCESS
```

### 2.4 部署

```bash
# 停止旧服务
net stop archive-backend

# 复制新 JAR
copy target\archive.jar D:\archive\apps\backend\archive.jar

# 启动新服务（生产 profile）
java -Xms512m -Xmx1536m -Dfile.encoding=UTF-8 -jar D:\archive\apps\backend\archive.jar --spring.profiles.active=prod

# 或通过 WinSW 启动
net start archive-backend
```

### 2.5 验证部署

```bash
# 健康检查（无需 token）
curl http://localhost:8080/api/health
# 期望: {"status":"UP"}

# 带 token 冒烟
TOKEN=$(curl -s -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"admin123"}' | python -c "import sys,json;print(json.load(sys.stdin)['data']['token'])")
curl -s http://localhost:8080/api/auth/me -H "Authorization: Bearer $TOKEN"
# 期望: 返回用户信息，无 500
```

## 3. 迁移说明

### 3.1 迁移脚本

| 文件 | 说明 |
|------|------|
| `deploy/sql/migrate_260611_01.sql` | **唯一**生产升级脚本（幂等，可重复执行） |
| `deploy/sql/init.sql` | **仅新库**重建用（含 DROP TABLE，生产勿跑） |

### 3.2 增量改动

本次升级包含以下数据库变更：

- 7 张表 ALTER（`deleted_at`/`deleted_by`/`version` 等）
- 10 张新表（`project_fact`、`notification`、`import_batch` 等）
- 6 个角色（`admin/committee/pm/user/legal/secretary`）
- 2 个 DB 触发器（`project_fact_event` INSERT-only）
- 回填 SQL（`confidence_level`、`audit_log.type`）

### 3.3 `spring.jpa.hibernate.ddl-auto`

- **生产**：使用 `application-prod.yml` → `ddl-auto: validate`（禁止 Hibernate 自动改表）
- **开发**：默认 `ddl-auto: update`（方便迭代）
- **切换方式**：启动时加 `--spring.profiles.active=prod`
