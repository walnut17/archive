# 07 — 安全机制

## 1. 热更新路径沙箱（deploy zip）

**目标**：`/v1/deploy/update` 收到 zip 后，**只能写入 `qa-agent/` 根目录内**文件，不能触及仓库其他目录或系统路径。

### 实现

| 模块 | 机制 |
|---|---|
| `app/services/path_sandbox.py` | `resolve_under_root()` — 解析后 `relative_to(qa_agent_root)` 校验 |
| `app/services/self_update.py` | zip 成员白名单 `app/tools/scripts` + `VERSION`；每文件写入前 `_safe_qa_path()` |
| 备份目录 | 仅 `qa-agent/.update_backup/` 内 |

### 拒绝示例

- `../backend/pom.xml`
- `qa-agent/../../config.json`
- zip 内绝对路径

## 2. LLM 脱敏

**目标**：调用外部 LLM **之前**对 prompt 脱敏；返回给用户前**还原**。覆盖姓名/机构名、证件号、金额。

### 实现

| 模块 | 机制 |
|---|---|
| `app/services/desensitize.py` | 正则识别 → 占位符 `[ORG_001]` `[PERSON_001]` `[ID_001]` `[AMT_001]` |
| `app/llm/glm.py` | `chat()` / `chat_stream()` 统一入口脱敏与流式还原 |

### 脱敏范围

- **机构/工厂名**：含「有限公司」「厂」「债权」等，如「岭兜建材二厂」「南安市岭兜建材二厂债权」
- **人名**：债务人/申请人/担保人等关键字后的 2–4 字
- **证件号**：身份证 15/18 位、统一社会信用代码、手机号
- **金额**：`300万元`、`545万`、`本金400万元` 等

### 配置

```json
"qaAgent": {
  "llmDesensitize": {
    "enabled": true
  }
}
```

环境变量覆盖：`QA_AGENT_LLM_DESENSITIZE_ENABLED=false`（通过 config 加载逻辑扩展时可加）

### 注意

- 引擎侧 **正则合成答案**（`react_helpers.py` 利率/抵押物）不经过 LLM，不脱敏
- `get_project_analysis` 读 DB 快照返回给用户为**原文**（数据已在库内，非外发 LLM）
- 后台 `analysis/extractor.py` 经 `glm_client`，**自动脱敏**
